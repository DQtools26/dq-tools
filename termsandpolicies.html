Privacy Policy

This website is designed with simplicity and privacy in mind. Many of the tools and features on the site are built using standalone HTML code embedded within pages hosted on Google Sites.

Because the site relies primarily on browser-based HTML tools, most functionality happens directly inside your device's web browser rather than on a remote server controlled by the site creator.

Data Collection

  • This website does not collect or store personal information.
  • No login system or account registration is required.
  • No personal data is saved by the HTML tools used on this website.

Local Processing

  • Many interactive tools operate directly within your browser.
  • Temporary data may exist only while the page is open.
  • All temporary data is automatically cleared when the page is refreshed, closed, or reloaded.

Offline Capability

  • Some features may continue functioning after the page loads, even without an active internet connection.
  • This is possible because the core functionality runs through local HTML scripts.

This design helps minimize privacy concerns because information entered into tools is not stored in a database, transmitted to external servers by the website itself, or permanently saved.

Please note that the website is hosted on Google Sites, which may operate under its own platform policies. Users should refer to Google's policies if they wish to learn more about how Google services handle general web traffic.

Overall, the intention of this site is to provide a safe, simple, and privacy-friendly environment where visitors can use tools and content without worrying about their data being saved.
